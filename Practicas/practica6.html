FORMULARIOS

Las etiquetas son:

Formulario

form -> declara el formulario

posee varios atributos, entre los cuales

action ->Es a la dirección donde se enviará el formulario

method -> Es el metodo por el cual se enviara la información, el GET es por URL y el POST es mas seguro

Inputs - Entradas de datos

input -> declaración de metodos de entrada

existen varios atributos para los inputs, entre los cuales se encuentran:

type -> Es el tipo de método de entrada

name -> Es el nombre del campo a enviar

placeholder -> Es el texto dentro del cuadro de texto como guía al usuario

required -> El atributo required indica al usuario que no puede avanzar sin llenar este campo

value -> aplica para los input del tipo submit, cambia el texto del botón
